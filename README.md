# Dynamic-Auxiliary-Guided-Unet-DAGUnet

Dynamic Auxiliary-Guided Unet (DAGUnet) is a deep-learning model for Arctic sea ice prediction

Currently, only a portion of the code related to DAGUnet has been uploaded; the complete code and data are currently being prepared.

All the training and testing data for DAGUnet used in this paper can be downloaded from the [[Zendo Link]](https://bhpan.buaa.edu.cn/link/AADBF25FF2D92B44C0BAAEAEB2135F4F76)
